ggplot2

  • Escrito por Hadley Wickham, quando ainda era um estudante de ps graduao;
  • um dos pacotes grficos mais utilizados na linguagem R, junto do pacote base e lattice;
  • a implementao do The Grammar of Graphics, idia desenvolvida por Leland Wilkinson.

ggplot2

Resumidamente, a gramtica nos diz que um grfico deve ser mapeado (mapping) a partir dos dados (data), de acordo com a aparncia (aesthetic) (color, formato, tamanho) de objetos geomtricos (geometric) (pontos, linhas, barras). A figura tambm pode conter transformaes estatsticas dos dados, sendo desenhada a partir de um sistema especfico de coordenadas.

Do livro "ggplot2"

ggplot2

  • Faa um paralelo com a nossa gramtica:
    • sujeito (data, aesthetics)
    • verbo (geometries, facets)
    • adjetivo (color, fill, shape, alpha)
  • Diminui a distncia da mente para a pgina.

Filosofia do ggplot2

  • Voc desenha, adicionando e modificando as camadas durante o processo.
    • No preciso realizar todo o desenho em um nico objeto (ao contrrio do lattice);
    • Uma vez que voc tenha adicionado uma entidade ao grfico, voc pode alterar ela posteriormente (ao contrrio do base);
    • Grficos complexos, com comandos simples (ao contrrio do lattice e base).

qplot

  • Exerccio 1: por que no precisamos especificar o geom?
qplot(x = conc, y = uptake, data = CO2)

qplot

  • Podemos adicionar elementos grficos, baseado em alguma informao sobre os dados.
qplot(x = conc, y = uptake, data = CO2, shape = Type, 
      size = 3, colour = Treatment)

qplot

  • Podemos criar painis diferentes com o argumento facets.
qplot(x = conc, y = uptake, data = CO2, facets = ~Type, 
      size = 3, colour = Treatment)

qplot

  • Tambm podemos alterar o ttulo dos eixos.
qplot(x = conc, y = uptake, data = CO2, facets = ~Type, 
      size = 3, colour = Treatment, 
      xlab = expression(paste("Concentrao de CO "[2])),
      ylab = expression(paste("Captura de CO "[2])))

qplot

  • Especificando outros tipos de geom, grficos diferentes podem ser feitos.
qplot(x = Treatment, y = uptake, data = CO2, facets = ~Type, 
      geom = "boxplot", fill = Treatment)

ggplot

  • a funo chave do ggplot2;
  • Permite customizao total do grfico.

ggplot

  • Aqui, a filosofia do The Grammar of Graphics explicitamente colocada em prtica.
  • Vamos comear desenhando a base do grfico: data e aesthetics.
g <- ggplot(data = CO2, mapping = aes(x = conc, y = uptake))
g

ggplot

  • Vamos separar os grficos de acordo com algumas categorias: facets.
g <- g + facet_grid(Treatment ~ Type, scales = "free_y")
g

ggplot

  • Vamos desenhar o que queremos no grfico: geom.
  • Vamos adicionar tambm como queremos o geom: colour, shape, alpha…
g <- g + geom_point(aes(colour = Treatment, shape = Type), size = 3, show.legend = FALSE)
g

ggplot

  • Podemos editar as cores dos pontos manualmente.
g <- g + scale_colour_manual(values = c("red", "blue"))
g

ggplot

  • Editar o ttulo dos eixos.
g <- g + xlab(expression(paste("Concentrao de CO "[2]))) +
  ylab(expression(paste("Captura de CO "[2])))
g

ggplot

  • E adicionar uma linha de tendncia.
g <- g + geom_smooth(linetype = 2, fill = "black", alpha = 0.1, colour = "black")
g

Interao com o dplyr

  • Voc pode realizar uma manipulao com um conjunto de dados, e encaminhar o ltimo formato da planilha de dados diretamente para o ggplot.
gapminder %>% 
  filter(country == "Brazil") %>% 
  ggplot(mapping = aes(x = year, y = gdpPercap)) +
  geom_point(aes(size = pop, colour = lifeExp), show.legend = FALSE) +
  geom_smooth() +
  theme_bw()

Interao com o dplyr

CO2 %>% 
  group_by(Type, Treatment) %>% 
  summarise(media_uptake = mean(uptake), desvio_uptake = sd(uptake),
            observacoes = n(), erro_uptake = desvio_uptake/sqrt(observacoes)) %>% 
  ggplot(aes(x = Type, y = media_uptake, fill = Treatment)) +
  geom_bar(stat = "identity", position = "dodge") +
  geom_errorbar(aes(ymin = media_uptake - erro_uptake,
                    ymax = media_uptake + erro_uptake),
                position = position_dodge(width = 0.9), width = 0.4)

Salvando um grfico do ggplot

  • A funo ggsave pode ser usada para salvar uma figura feita no ggplot2, em qualquer formato que voc desejar;
  • Voc tambm pode usar as funes disponveis no pacote base, como pdf, tiff e jpeg;
  • Para grficos com mais de um painel, voc pode usar a funo grid.arrange para criar o grfico de painel, e depois usar as funes do pacote base para salv-las.
ggsave(filename = "/Users/Nick/Dropbox/R PPGE_UFRJ/3 - Quarta/figures/figura.png", 
       plot = figura, width = 15, height = 10, units = "cm", dpi = 200)
